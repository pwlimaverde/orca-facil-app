# Instruções para Coleta de Evidências Visuais (Screenshots)

Para completar a documentação do projeto, por favor, execute o aplicativo no emulador ou dispositivo físico e capture as seguintes telas. Salve as imagens nesta pasta (`doc/evidencias`) com os nomes sugeridos abaixo (formato .png ou .jpg).

## Lista de Screenshots Necessários

1.  **Tela Principal (Home)**
    *   **O que mostrar:** A lista de orçamentos cadastrados (ou a tela vazia se não houver).
    *   **Nome do arquivo:** `tela_home.png`

2.  **Tela de Cadastro de Orçamento**
    *   **O que mostrar:** O diálogo ou tela onde se insere o nome e descrição do novo orçamento.
    *   **Nome do arquivo:** `tela_cadastro_orcamento.png`

3.  **Tela de Detalhes do Orçamento**
    *   **O que mostrar:** A tela mostrando o orçamento criado, com a lista de itens (vazia ou preenchida) e o valor total.
    *   **Nome do arquivo:** `tela_detalhes_orcamento.png`

4.  **Tela de Adição de Item**
    *   **O que mostrar:** O diálogo ou formulário para adicionar um produto/serviço ao orçamento.
    *   **Nome do arquivo:** `tela_cadastro_item.png`

---
**Nota:** Os trechos de código (`snippet_logica_calculo.kt` e `snippet_persistencia_room.kt`) já foram extraídos automaticamente e estão salvos nesta pasta.
